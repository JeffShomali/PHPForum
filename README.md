# PHPForum
This is a simple Object Oriented PHP forum application.

## Technology Used :
`HTML`, `CSS`, `Bootstrap`, `Javascript`, `jQuery`, `PHP and MySQL (PDO)`

## Features
- Object Oriented PHP
- Simple MVC
- Simple Authentication
- Post a topic
- Add a new category
- Replies to topic
